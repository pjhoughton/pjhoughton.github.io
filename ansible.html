‚ùì What is Ansible?

Ansible is an open-source IT automation engine that automates provisioning, configuration management, application deployment, orchestration, and many other IT processes. It is free to use and benefits from thousands of contributors.

Its main strengths are simplicity and ease of use. It also emphasizes security and reliability, featuring minimal moving parts. Ansible uses OpenSSH for transport and a human-readable language designed for quick adoption.

Key Concepts

  • Control Node: The machine from which you run the Ansible CLI tools.
  • Managed Nodes: Target devices (servers, appliances, or computers) managed by Ansible.
  • Inventory: A list of managed nodes, grouped and assigned variables.
  • Playbooks: YAML files containing plays that define automation tasks.
  • Plays: Map hosts to tasks, variables, and roles in an ordered list.
  • Roles: Reusable collections of tasks, handlers, variables, and templates.
  • Tasks: Individual actions applied to managed hosts.
