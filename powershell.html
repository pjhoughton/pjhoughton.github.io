‚ùì What is PowerShell?

PowerShell is a task automation and configuration management framework from Microsoft. It combines a command‚Äëline shell with a powerful scripting language built on .NET. PowerShell is designed to help system administrators and engineers automate tasks, manage systems, and integrate with modern platforms.

Unlike traditional shells, PowerShell works with objects rather than plain text, making it easier to manipulate structured data and interact with complex systems.

Windows PowerShell vs PowerShell (Core)

  • Windows PowerShell: The original version, built on the full .NET Framework. It is Windows‚Äëonly and deeply integrated with Windows management tools.
  • PowerShell (Core): The modern, cross‚Äëplatform version built on .NET Core. It runs on Windows, Linux, and macOS, making it more versatile for hybrid environments.

Key Concepts

  • Cmdlets: Lightweight commands built into PowerShell (e.g., Get-Process).
  • Modules: Packages of cmdlets and functions that extend PowerShell‚Äôs capabilities.
  • Objects: PowerShell outputs structured objects, not plain text, enabling advanced manipulation.
  • Pipelines: Allow chaining commands together, passing objects between them.
  • Scripts: Collections of commands saved in .ps1 files for reuse.
  • Remoting: Execute commands on remote systems using Enter-PSSession or Invoke-Command.
